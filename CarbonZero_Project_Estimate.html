Risks & Mitigations
⚠️CrewAI Agent Latency
High

Groq Llama-3 multi-agent chains can run 15–45s. Without streaming or progress indicators, UX degrades sharply. Mitigation: add async task queuing and Streamlit spinner skeletons per agent call.

⚠️Emission Factor Accuracy
High

Hardcoded EF_GRID (0.716 vs 0.82 between files) inconsistency exists. India CEA factors change annually. Mitigation: centralise all factors in emission_factors.py, add version/source metadata.

🔶Streamlit State Management
Medium

JSON flat-file storage will hit concurrency limits with multiple users. Mitigation: migrate to SQLite or Supabase Postgres before beta, keeping the existing schema.

🔶CBAM Regulation Changes
Medium

EU CBAM thresholds and covered sectors are evolving. The regulation_radar agent relies on LLM knowledge, not live feeds. Mitigation: add a web_search tool to the regulation agent or a quarterly data update process.

🟢Groq API Rate Limits
Low

Free tier Groq limits are generous for SME single-tenant usage. Mitigation: implement request caching for identical emission summaries and exponential back-off on 429s.

🟢PDF Unicode / Font Gaps
Low

report_generator.py has a DejaVu font fallback but the fonts/ directory isn't in the repo. Mitigation: bundle DejaVuSans.ttf or switch to ReportLab which bundles fonts natively.

Strategic Guidance
Recommendations
01
Unify Emission Factors — One Source of Truth

data_generator.py uses EF_GRID = 0.716 while emission_factors.py has 0.82 for India Grid. This creates silent calculation divergence between the demo data generator and the live app. Centralise all factors in emission_factors.py and import from it everywhere. Estimated fix: 4 hrs.

Bug / Data Integrity
02
Replace JSON Flat File Storage with SQLite

data/emissions.json will break under concurrent writes and grows unbounded. A local SQLite database via SQLAlchemy is a zero-cost, zero-infrastructure swap that adds transactions, indexing, and proper queries. Migration preserves the existing schema 1:1.

Architecture
03
Add Streaming Output to AI Agents

The five CrewAI agents currently block until completion with no visible progress. Using Streamlit's st.write_stream() or a custom token-streaming callback will dramatically improve perceived performance for report generation and offset advisor tasks (typically 10–30s).

UX / Performance
04
Bundle DejaVuSans.ttf in the Repository

report_generator.py has a full Unicode fallback path that loads fonts/DejaVuSans.ttf, but this directory is absent from the uploaded codebase. When missing, all Devanagari/special characters in company names silently corrupt. Add the font file and include it in requirements/setup docs.

Bug
05
Scope the BRSR Automation Module as a Paid Add-On

BRSR (Business Responsibility and Sustainability Reporting) is mandatory for India's top 1,000 listed companies and increasingly adopted by SME supply chains. A structured BRSR data-to-template export would differentiate CarbonZero commercially and justify a SaaS pricing tier.
